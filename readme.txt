== Description ==

VintageJS allows you to apply a custom retro, vintage look to WordPress post images.

This jQuery-Plugin was tested and worked fine in the following browsers:

Mozilla FireFox 3.16.14
Google Chrome 9.0.597.107
Apple Safari 5.0.3
Opera 11.01
Internet Explorer 9

Javascript by R. Fleischmann http://vintagejs.com


== Installation ==

= Automatic Installation =

1. From inside your WordPress administration panel, visit 'Plugins -> Add New'
2. Search for `Vintage.js` and find this plugin in the results
3. Click 'Install'
4. Once installed, activate via the 'Plugins -> Installed' page
5. Choose various admin options.

= Manual Installation =

1. Upload `vintagejs` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
5. Choose various admin options.

== Instructions ==
To add vintage effect to an image you just need to add the class vintage to an uploaded image. After inserting image into the post editor click image and choose the image edit icon. Then choose 'Advanced Settings'. At the end of the CSS class add the word vintage. Click update and then update the post. By adding the class vintage to an image you are telling the plugin to add the retro effect to the individual image.
